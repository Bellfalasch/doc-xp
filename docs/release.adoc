== Upgrading

Enonic XP {release} contains breaking changes.
As such, customers planning to migrate from version 6.x must complete two steps rather than one.

1. <<upgrade-xp#,Upgrading Enonic XP>>
2. <<upgrade-apps#,Upgrading installed applications>>

Follow the instructions carefully to secure a successful upgrade.
